About Me

Background

After exploring front-end development on Free Code Camp, an online curriculum for web development, I knew that coding was my passion -- a perfect mix of logic and creativity. Skills & Technologies

Front-End

The front end technologies that I've worked with are HTML5, CSS3, JavaScript, and jQuery. I've used these technologies in projects as standalone sites and within Rails applications.

Responsive Design

I've gotten to work with a number of responsive design techniques and technologoies including: Bootstrap, FlexBox, and Media Queries. Its important to me to make applications that look good on both mobile and desktop clients.

Back-end

Ruby, Ruby on Rails (4 and 5), and Postgres are the primary technologies I've worked with from a back-end perspective. For me, Ruby provides a consistent, efficient, and intelligent way to develop web applications from the ground up, but I'm always looking to learn new technologies.

Testing

At LEARN its not done until its tested. We've practiced both Test Driven Design (TDD), Behavior Driven Design (BDD) using Rspec and Capybara. Although its not always my favorite part of development, as an ex-auditor, I understand its importance and value in any application.

APIs

APIs are one of my favorite things I've learned so far because they give you access to the rest of the internet world. I've worked with Twitter's tweet and authentication APIs, Facebook's authentication API, the GeoLocation API, Google's Map API, IP Info's API, Open Weather Map's API and more.

Release Technologies

As part of the development and release process I've gained valuable experience with Git, GitHub, Heroku, AWS S3, GitHub Pages, and CodePen. I've used each of these technologies in mulptiple projects to help track, deploy, and manage the applications I've developed.
